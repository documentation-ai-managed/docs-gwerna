## Overview

Gwerna helps you create, organize, and manage documentation spaces for your projects. You build structured knowledge bases with version control, search, and collaboration tools. Focus on content while Gwerna handles hosting, SEO, and updates.

<Callout kind="info">
  Gwerna integrates with your Git workflow, so you edit docs in Markdown and publish automatically.
</Callout>

## Quick Start

Get up and running in minutes.

<Steps>
  <Step title="Sign Up" icon="user-plus">
    Visit `https://gwerna.com` and create a free account with your email or GitHub.
  </Step>
  <Step title="Create a Space" icon="plus">
    Click "New Space" and name it, like "My Project Docs".
  </Step>
  <Step title="Add Your First Doc" icon="file-text">
    Upload Markdown files or use the editor. Preview changes live.

````mdx
# Welcome to My Docs

## Quick Install

```bash
npm install my-package
```
````

  </Step>
  <Step title="Publish" icon="globe">
    Hit publish to generate your site at `https://docs.gwerna.com/my-space`.
  </Step>
</Steps>

## Basic Terminology

<ExpandableGroup>
  <Expandable title="Space" default-open="true">
    A Space is your isolated documentation site. Create one per project or product.
  </Expandable>
  <Expandable title="Doc">
    Individual Markdown or MDX pages within a Space. Supports components like `<Callout>` and code blocks.
  </Expandable>
  <Expandable title="Version">
    Track changes with Git-like versioning. Roll back or branch docs as needed.
  </Expandable>
</ExpandableGroup>
