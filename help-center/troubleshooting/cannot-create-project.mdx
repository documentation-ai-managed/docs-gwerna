---
title: Cannot Create New Project
description: Resolve issues creating new projects in Gwerna.
---

## Symptoms

- The **Create** button is disabled.
- Error: "Project creation failed."

## Common Causes

- Project name left blank
- Network connectivity issues
- Exceeded project limit for your plan

## Solution

<Steps>
  <Step title="Check project name">
    Make sure you entered a valid, unique name.
  </Step>
  <Step title="Test connection">
    ````bash
    ping app.gwerna.com
    ````
  </Step>
  <Step title="Upgrade plan if needed">
    Visit **Account > Billing** to check your plan and upgrade if necessary.
  </Step>
</Steps>
